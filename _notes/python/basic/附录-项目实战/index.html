Python 项目实战¶

项目一：智能温度转换器¶

核心知识点：变量、数字运算、字符串格式化、input()、if-elif-else、函数、异常处理

项目目标¶

编写一个命令行工具，让用户输入温度和单位（摄氏度/华氏度），程序自动转换为另一种单位并输出。同时能处理错误输入。

功能需求¶

  • 提示用户输入温度值（如 100）。
  • 提示用户输入单位（C 或 F）。
  • 根据单位进行转换：
  • 摄氏度 → 华氏度：F = C * 9/5 + 32
  • 华氏度 → 摄氏度：C = (F - 32) * 5/9
  • 输出结果，保留一位小数。
  • 如果输入非法（非数字、单位错误），给出友好提示。

分步实现¶

第一步：创建文件并编写基础结构 新建 temp_converter.py，先写出获取用户输入的逻辑。

# 第一步：获取原始输入
temp_input = input("请输入温度值: ")
unit = input("请输入单位（C 或 F）: ").upper()
print(f"你输入了 {temp_input} {unit}")

运行测试：输入 100 c，观察输出。

第二步：将温度字符串转换为浮点数（带异常处理） 用户可能输入 abc，所以用 try-except 包裹转换。

try:
    temp = float(temp_input)
except ValueError:
    print("错误：温度值必须是数字。")
    exit(1)   # 用非零值退出程序，表示异常

将上面的代码整合入文件，再次测试输入 abc，应该看到错误提示。

第三步：实现转换逻辑 使用 if-elif-else 判断单位，并计算。

if unit == 'C':
    converted = temp * 9 / 5 + 32
    target_unit = 'F'
elif unit == 'F':
    converted = (temp - 32) * 5 / 9
    target_unit = 'C'
else:
    print("错误：单位只能输入 C 或 F。")
    exit(1)

print(f"{temp:.1f}°{unit} = {converted:.1f}°{target_unit}")

完整程序此时应该能正确转换。测试用例：0 C → 32.0 F，212 F → 100.0 C。

第四步：封装成函数 把核心转换逻辑提取为一个函数，提高复用性。

def convert_temperature(temp, unit):
    if unit == 'C':
        return temp * 9 / 5 + 32, 'F'
    elif unit == 'F':
        return (temp - 32) * 5 / 9, 'C'
    else:
        raise ValueError("单位必须为 C 或 F")

然后在主程序里调用：

try:
    new_temp, new_unit = convert_temperature(temp, unit)
    print(f"{temp:.1f}°{unit} = {new_temp:.1f}°{new_unit}")
except ValueError as e:
    print(f"错误：{e}")
    exit(1)

再次测试各项输入，确保运行正常。

第五步：添加循环，允许连续转换（可选） 如果想继续巩固循环，可以让用户一次输入多个值，输入 q 退出。

while True:
    cmd = input("输入温度或 q 退出: ").strip()
    if cmd.lower() == 'q':
        break
    unit = input("单位 C/F: ").strip().upper()
    try:
        temp = float(cmd)
        new_temp, new_unit = convert_temperature(temp, unit)
        print(f"{temp:.1f}°{unit} = {new_temp:.1f}°{new_unit}")
    except ValueError as e:
        print(f"错误：{e}")

至此，你完成了一个健壮的小工具，它融合了基础语法、条件判断、类型转换、异常处理和函数。

项目二：通讯录管理器（命令行版）¶

核心知识点：列表、字典、循环、函数、文件读写（JSON 持久化）、异常处理

项目目标¶

构建一个命令行通讯录，支持添加、查找、删除、列出所有联系人，并在退出时把数据保存到 JSON 文件，下次启动自动加载。

功能需求¶

  • 启动时从 contacts.json 读取已有联系人（如果文件不存在则创建空字典）。
  • 提供菜单：1.添加 2.查找 3.删除 4.显示全部 5.退出。
  • 联系人信息以姓名为键，值为字典，存电话和邮箱。
  • 退出时将数据写回 JSON 文件。

分步实现¶

第一步：创建脚本和加载/保存函数 新建 contact_book.py。首先导入 json 和 os，实现加载和保存。

import json
import os

FILENAME = "contacts.json"

def load_contacts():
    if os.path.exists(FILENAME):
        with open(FILENAME, 'r', encoding='utf-8') as f:
            return json.load(f)
    return {}

def save_contacts(contacts):
    with open(FILENAME, 'w', encoding='utf-8') as f:
        json.dump(contacts, f, ensure_ascii=False, indent=2)

测试：运行脚本，检查是否会生成文件，并能手工写入一些数据验证读取。

第二步：添加联系人功能 用函数实现，传入总字典。

def add_contact(contacts):
    name = input("姓名: ").strip()
    if not name:
        print("姓名不能为空")
        return
    phone = input("电话: ").strip()
    email = input("邮箱: ").strip()
    contacts[name] = {"phone": phone, "email": email}
    print(f"已添加联系人: {name}")

第三步：查找联系人 根据姓名精确匹配。

def find_contact(contacts):
    name = input("要查找的姓名: ").strip()
    if name in contacts:
        info = contacts[name]
        print(f"姓名: {name}, 电话: {info['phone']}, 邮箱: {info['email']}")
    else:
        print(f"未找到联系人: {name}")

第四步：删除联系人 用 pop 方法，找不到则提示。

def delete_contact(contacts):
    name = input("要删除的姓名: ").strip()
    if name in contacts:
        del contacts[name]
        print(f"已删除联系人: {name}")
    else:
        print(f"未找到联系人: {name}")

第五步：显示全部联系人 遍历字典，格式化输出。

def show_all(contacts):
    if not contacts:
        print("通讯录为空")
        return
    print("\n所有联系人:")
    for name, info in contacts.items():
        print(f"  {name}: 电话 {info['phone']}, 邮箱 {info['email']}")

第六步：主循环菜单 用一个 while 循环加 if-elif 分支控制程序流程。

def main():
    contacts = load_contacts()
    while True:
        print("\n--- 通讯录 ---")
        print("1. 添加联系人")
        print("2. 查找联系人")
        print("3. 删除联系人")
        print("4. 显示全部")
        print("5. 退出")
        choice = input("请选择 (1-5): ").strip()
        if choice == '1':
            add_contact(contacts)
        elif choice == '2':
            find_contact(contacts)
        elif choice == '3':
            delete_contact(contacts)
        elif choice == '4':
            show_all(contacts)
        elif choice == '5':
            save_contacts(contacts)
            print("已保存，再见！")
            break
        else:
            print("无效选择，请重试")

if __name__ == "__main__":
    main()

第七步：添加异常防护与优化

  • 添加联系人时，可检查是否已存在，提示是否覆盖。
  • 处理 json.decoder.JSONDecodeError（如果文件损坏，重新初始化）。
  • 在 load_contacts 中添加 try-except：
def load_contacts():
    if os.path.exists(FILENAME):
        try:
            with open(FILENAME, 'r', encoding='utf-8') as f:
                return json.load(f)
        except json.JSONDecodeError:
            print("通讯录文件损坏，将重建。")
    return {}

这样你的通讯录工具就完工了。你用它管理数据，不知不觉练习了字典、文件、异常处理和整个命令行应用的结构。

项目三：文本词频统计器¶

核心知识点：文件读取、字符串处理、字典计数、排序、collections.Counter、生成器、argparse（初探）

项目目标¶

读取一个文本文件，统计每个单词出现的次数，按频率从高到低输出前 N 个词。支持通过命令行参数指定文件和显示数量。

功能需求¶

  • 读取指定文本文件，忽略大小写，移除标点符号。
  • 统计所有单词的频率。
  • 输出前 N 个高频词及其次数。

分步实现¶

第一步：处理命令行参数 用 sys.argv 或 argparse 最小化实现。先用简单方式：脚本名后接文件路径，可选的第二个参数是显示数量（默认 20）。让初学者练习 sys.argv。

import sys

def parse_args():
    if len(sys.argv) < 2:
        print("用法: python word_counter.py <文件路径> [显示数量]")
        sys.exit(1)
    filename = sys.argv[1]
    top_n = int(sys.argv[2]) if len(sys.argv) > 2 else 20
    return filename, top_n

第二步：读取文件并清洗文本 定义函数 read_and_clean(filename)，利用 with open 和生成器逐行处理，避免一次性全部加载大文件。

import string

def read_words(filename):
    with open(filename, 'r', encoding='utf-8') as f:
        for line in f:
            # 去除标点，转小写
            line = line.translate(str.maketrans('', '', string.punctuation))
            for word in line.lower().split():
                yield word

这里使用了 yield，让函数变成一个生成器，惰性产出单词，内存效率极高。

第三步：统计词频 使用普通字典或 collections.Counter。先用 Counter 简化并展示标准库的威力。

from collections import Counter

def count_words(filename):
    word_counts = Counter()
    for word in read_words(filename):
        word_counts[word] += 1
    return word_counts

或者直接 Counter(read_words(filename))，因为 Counter 可以接受可迭代对象。

第四步：排序输出 利用 most_common 方法直接得到频率降序列表。

def main():
    filename, top_n = parse_args()
    try:
        counter = count_words(filename)
    except FileNotFoundError:
        print(f"错误：文件 {filename} 不存在。")
        sys.exit(1)
    for word, count in counter.most_common(top_n):
        print(f"{word}: {count}")

第五步：集成并添加 __main__ 守护 完整脚本如下：

import sys
import string
from collections import Counter

def parse_args():
    if len(sys.argv) < 2:
        print("用法: python word_counter.py <文件> [显示数量]")
        sys.exit(1)
    return sys.argv[1], int(sys.argv[2]) if len(sys.argv) > 2 else 20

def read_words(filename):
    with open(filename, 'r', encoding='utf-8') as f:
        for line in f:
            line = line.translate(str.maketrans('', '', string.punctuation))
            for word in line.lower().split():
                yield word

def count_words(filename):
    return Counter(read_words(filename))

def main():
    filename, top_n = parse_args()
    try:
        counter = count_words(filename)
    except FileNotFoundError:
        print(f"错误: 文件 {filename} 不存在。")
        sys.exit(1)
    for word, count in counter.most_common(top_n):
        print(f"{word}: {count}")

if __name__ == "__main__":
    main()

第六步：测试与扩展

  • 找一篇英文文章（或中文但需分词，可引入 jieba 库作扩展），测试输出。
  • 可改进清洗逻辑：用正则去掉数字或单字母单词。
  • 使用 argparse 改造参数解析，体验更专业的命令行接口。

通过这个项目，你深入实践了文件处理、生成器、Counter 和简单的脚本组织。

项目四：简易学生管理系统（面向对象 + CSV 持久化）¶

核心知识点：面向对象（类、属性、方法、__str__）、继承、异常处理、模块组织、CSV 文件读写、列表操作、排序

项目目标¶

用面向对象设计一个学生管理系统，学生数据存于 CSV 文件。支持添加学生、显示全部、按成绩排序、删除学生，启动时自动加载，退出时自动保存。

功能需求¶

  • Student 类：包含姓名、学号、成绩属性；实现 __str__ 用于美观打印。
  • StudentManager 类：管理学生列表，负责文件读写、增删查改和排序。
  • 主程序提供循环菜单。
  • 数据存储为 students.csv，每行：学号,姓名,成绩。

分步实现¶

第一步：定义 Student 类 创建 models.py，把数据类放进去，符合模块化思想。

# models.py
class Student:
    def __init__(self, student_id, name, score):
        self.student_id = student_id
        self.name = name
        self.score = score

    def __str__(self):
        return f"[{self.student_id}] {self.name} - {self.score}分"

第二步：定义 StudentManager 类 创建 manager.py，实现核心逻辑。

# manager.py
import csv
import os
from models import Student

FILENAME = "students.csv"

class StudentManager:
    def __init__(self):
        self.students = []  # 存放 Student 对象的列表

    def load_from_file(self):
        if not os.path.exists(FILENAME):
            return
        with open(FILENAME, 'r', encoding='utf-8') as f:
            reader = csv.reader(f)
            for row in reader:
                if len(row) == 3:  # 简单校验
                    sid, name, score_str = row
                    try:
                        score = float(score_str)
                    except ValueError:
                        score = 0.0
                    self.students.append(Student(sid, name, score))

    def save_to_file(self):
        with open(FILENAME, 'w', newline='', encoding='utf-8') as f:
            writer = csv.writer(f)
            for stu in self.students:
                writer.writerow([stu.student_id, stu.name, stu.score])

    def add_student(self, sid, name, score):
        # 检查学号唯一性
        if any(s.student_id == sid for s in self.students):
            raise ValueError(f"学号 {sid} 已存在")
        stu = Student(sid, name, score)
        self.students.append(stu)

    def remove_student(self, sid):
        for i, stu in enumerate(self.students):
            if stu.student_id == sid:
                del self.students[i]
                return True
        return False

    def show_all(self):
        if not self.students:
            print("（无学生数据）")
            return
        for stu in self.students:
            print(stu)

    def sort_by_score(self, reverse=False):
        self.students.sort(key=lambda s: s.score, reverse=reverse)

    def find_student(self, sid):
        for stu in self.students:
            if stu.student_id == sid:
                return stu
        return None

第三步：编写主程序调用这些类 创建 main.py。

# main.py
from manager import StudentManager

def main():
    mgr = StudentManager()
    mgr.load_from_file()
    while True:
        print("\n===== 学生管理系统 =====")
        print("1. 添加学生")
        print("2. 显示所有")
        print("3. 按成绩排序显示")
        print("4. 删除学生")
        print("5. 查找学生")
        print("6. 退出")
        choice = input("选择: ").strip()
        try:
            if choice == '1':
                sid = input("学号: ").strip()
                name = input("姓名: ").strip()
                score = float(input("成绩: "))
                mgr.add_student(sid, name, score)
                print("添加成功。")
            elif choice == '2':
                mgr.show_all()
            elif choice == '3':
                mgr.sort_by_score(reverse=True)
                print("按成绩降序：")
                mgr.show_all()
            elif choice == '4':
                sid = input("要删除的学号: ").strip()
                if mgr.remove_student(sid):
                    print("删除成功。")
                else:
                    print("未找到该学号。")
            elif choice == '5':
                sid = input("查找学号: ").strip()
                stu = mgr.find_student(sid)
                if stu:
                    print(stu)
                else:
                    print("未找到。")
            elif choice == '6':
                mgr.save_to_file()
                print("数据已保存，再见！")
                break
            else:
                print("无效选择。")
        except ValueError as e:
            print(f"错误: {e}")
        except Exception as e:
            print(f"发生未知错误: {e}")

if __name__ == "__main__":
    main()

第四步：运行测试

  • 添加几个学生，退出后再启动，观察加载是否正常。
  • 尝试添加重复学号，应弹出错误。
  • 删除学生并确认。
  • 按成绩排序查看效果。

第五步：拓展建议

  • 使用 @property 为 score 添加范围校验（0-100）。
  • 增加修改学生信息的功能。
  • 用 pickle 或 json 替换 CSV 作为存储。
  • 将 Manager 进一步抽象为基类，然后用 JSONManager、CSVManager 继承实现多态存储。
