## 📚 Project Introduction

This is a personal technical note repository built with MkDocs + Material theme, used to record technical summaries, practical experiences and thoughts during the learning process.

## ✨ Features

- 🎨 **Modern Interface**: Material Design style, supports automatic dark/light mode switching
- 🔍 **Full-text Search**: Built-in powerful search function for quick content location
- 📱 **Responsive Design**: Perfectly adapts to desktop and mobile devices
- 🚀 **Quick Deployment**: Integrated with GitHub Actions for automatic deployment to GitHub Pages
- 📝 **Markdown Writing**: Use concise Markdown syntax, focus on content creation

## 🚀 Quick Start

### Environment Requirements

- Python 3.8+
- MkDocs 1.4+
- MkDocs Material 9.0+

### Local Development

```bash
# Clone the repository
git clone https://github.com/duan-deqing/notes.git

# Enter the project directory
cd notes

# Install dependencies
pip install -r requirements.txt

# Start development server
mkdocs serve --livereload

# Visit http://localhost:8000
```
