# 变量与基本数据类型

## 变量定义与命名规范

### 变量定义

> 变量可以理解为一个 **带名字的储物箱**。你把数据放进箱子，贴上标签，之后就可以通过标签来取用或更换里面的东西。在 Python 中创建一个变量极其简单：你甚至不需要声明类型，赋值的那一刻它就诞生了。

```python
name = "Ada"
age = 25
pi = 3.14159
```

!!! note
    程序运行时，Python 解释器会在内存中创建对应的对象（一个字符串 "Ada"、一个整数 25、一个浮点数 3.14159），然后让变量名分别指向它们。
    这就是 **动态类型**：变量名本身没有固定类型，它只是指向对象的标签。
    ```python
    x = 10      # x 指向一个整数
    x = "hello" # 现在 x 指向一个字符串，完全合法
    ```

!!! warning "命名规范（违反会直接报错）"
    起名字是程序员的日常，Python 对此有严格的语法约束和公认的风格约束。
    - 只能包含 : 字母`（a-z, A-Z）`、数字`（0-9）`和下划线 `_`。
    - 不能以数字开头 : `1st_place` 非法，`first_place` 合法。
    - 严格区分大小写 : `name、Name、NAME` 是三个完全不同的人。
    - 不能使用 Python 的保留关键字 : 这些词被语言本身占用
    !!! note "例如"
        `if、for、while、class、True、False、None、and、or、not` 等。
        你可以用 `import keyword` `print(keyword.kwlist)` 查看完整列表。

!!! tip "风格规则（违反不会报错，但同行会皱眉）"
    - **蛇形命名法（snake_case）**：变量名和函数名使用全小写字母，单词之间用下划线连接。例如 `user_name`、`total_price`。这是 PEP 8 的推荐。
    - **清晰胜于短小**：cnt 不如 count，d 不如 days。代码是写给人看的。
    - 避免使用单字母 l（小写 L）、O（大写 o）、I（大写 i）作为变量名，它们在很多字体中与数字 1、0 难以分辨。
    - 双下划线开头和结尾的名字（如 `__init__`）是 Python 的“魔法方法”，不要自己发明这类名字，除非你知道自己在做什么。
    ```python
    # 好例子
    max_students = 30
    total_score = 95.5

    # 差例子（能运行，但可读性差）
    a = 30
    a1 = 95.5
    x2k = "hello"   # 这到底在表达什么？
    ```

### * 多重赋值与交换

Python 允许同时给多个变量赋值：

```python
x, y, z = 10, 20, "hello"   # x=10, y=20, z="hello"
```

这其实是 **元组解包** 的语法糖，背后原理后面会讲。利用这个特性，交换两个变量的值不需要临时变量：

```python
a, b = b, a   # a 和 b 的值互换
```

### * 变量的底层模型

> 用一个通俗的比喻：变量名像是贴在对象上的便利贴，而不是对象本身。同一个对象可以被多个变量名引用。

```python
a = [1, 2, 3]
b = a       # b 也指向同一个列表对象
b.append(4)
print(a)    # [1, 2, 3, 4] —— a 也被改变了
```

但对于不可变类型（整数、字符串等），赋值实际上是重新贴标签到新对象上，原对象不会变。这个细节会在后续深入讨论。

## **数字类型: int、float、complex**

### 整数 int

Python 3 中的 int 是任意精度的，也就是说它能表示多大的数字，只取决于你计算机的内存，不存在“溢出”问题。

```python
small = 42
large = 123456789012345678901234567890
negative = -7
```

数字中间可以用下划线分隔，提高可读性：

```python
million = 1_000_000      # 一百万，Python 3.6+ 支持
bignum = 0x_FF_EE_DD     # 十六进制同样可用
```

!!! note "常见进制表示"
    - 十进制：直接写数字 `255`
    - 二进制：0b 前缀，`0b11111111`
    - 八进制：0o 前缀，`0o377`
    - 十六进制：0x 前缀，`0xFF`
    - 使用 `int()` 函数可以将其他类型转换为整数，比如将字符串转换为整数：
    ```python
    int("42")        # 42
    int("1010", 2)   # 10，指定基数为2，把二进制字符串转成十进制
    int(3.14)        # 3，截断小数部分，不是四舍五入
    ```

### 浮点数 float

浮点数是有小数点的数字，底层使用 IEEE 754 双精度（64 位）表示。
这意味着它能表示非常大和非常小的数，但精度有限——约 15-17 位有效十进制数字。

```python
pi = 3.141592653589793
radius = 2.0
```

!!! warning "精度陷阱"
    由于二进制无法精确表示某些十进制小数，你可能看到意料之外的尾巴：
    ```python
    0.1 + 0.2   # 结果是 0.30000000000000004 而不是 0.3
    ```
    这不是 Python 的 bug，而是浮点数的先天特性。在处理货币等需要精确小数的场景，应使用 `decimal` 模块。

浮点数支持科学计数法：

```python
avogadro = 6.02e23      # 6.02 × 10²³
electron_mass = 9.11e-31 # 9.11 × 10⁻³¹
```

float() 函数可将整数、数字字符串转为浮点数：

```python
float(42)       # 42.0
float("3.14")   # 3.14
```

### * 复数 complex

Python 原生支持复数，这是工程和科学计算的加分项。虚数单位用 j 表示（有时大写 J）。

```python
z = 2 + 3j
w = complex(1, -2)   # 1 - 2j

# 可以分别访问实部和虚部
z.real    # 2.0
z.imag    # 3.0
# 复数运算遵循数学规则
(1 + 2j) * (3 + 4j)   # (-5+10j)
abs(3 + 4j)           # 5.0，模长
```

## 算术运算符与运算优先级

Python 提供了丰富且直观的算术运算符。

| 运算符 | 描述                   | 示例     | 结果  |
| :----- | :--------------------- | :------- | :---- |
| `+`    | 加法                   | `5 + 2`  | `7`   |
| `-`    | 减法                   | `5 - 2`  | `3`   |
| `*`    | 乘法                   | `5 * 2`  | `10`  |
| `/`    | 除法（结果总为浮点数） | `5 / 2`  | `2.5` |
| `//`   | 地板除（向下取整）     | `5 // 2` | `2`   |
| `%`    | 取模（余数）           | `5 % 2`  | `1`   |
| `**`   | 幂运算                 | `5 ** 2` | `25`  |

几个需要留意的点：

- `/` 除法在 Python 3 中 **总是** 返回浮点数，哪怕结果刚好是整数：`4 / 2` 得到 `2.0`。如果你想要整数结果，用 `//`。
- `//` 地板除向负无穷方向取整：`-5 // 2` 结果是 `-3`（因为 `-2.5` 向下取整是 `-3`），这与某些其他语言的“向零截断”不同。
- 取模 `%` 的结果总是与被除数符号一致：`-5 % 2` 得到 `1`（因为 `-5 = -3 * 2 + 1`）。
- 幂运算 `**` 是右结合的：`2 ** 3 ** 2` 等价于 `2 ** 9` 即 `512`，不是先算左。

!!! warning "运算优先级"
    从高到低：
    1. `**` （幂，右结合）
    2. `+x, -x` （正负号，一元）
    3. `*, /, //, %` （乘除取模）
    4. `+, -` （加减，二元）
    同优先级从左往右。当记不住或有歧义时，**直接上括号**。括号明确意图，也让读代码的人省心：
    ```py
    total = (price + tax) * quantity   # 清晰无误
    ```
    Python 支持增强赋值运算符：
    ```python
    x = 10
    x += 3   # 等价于 x = x + 3
    x *= 2   # 等价于 x = x * 2
    ```
    这种写法既简洁，有时执行效率也稍高（某些对象能原地修改）。

## 字符串定义、转义与原始字符串

> 字符串用来处理文本，是 Python 中最常用的类型之一。它的本质是 **不可变的 Unicode 字符序列**。

### 定义字符串

可以使用成对的单引号、双引号，或三个引号（单双皆可）,单引号与双引号完全等效，选哪种主要是习惯。

```py
s1 = 'Hello'
s2 = "Hello"
s3 = '''这是一段
多行文本'''
s4 = """也可以
这样写"""
```

!!! tip "一个小技巧"
    如果字符串内含有引号，可以用另一种引号包裹，避免了转义
    ```python
    quote = "He said, 'Python is awesome!'"
    sentence = 'Let\'s go'     # 使用转义
    sentence2 = "Let's go"     # 更清爽
    ```

三引号可以跨越多行，常用来写文档字符串（docstring）或多行长注释。

### 转义字符

反斜杠 \ 用于引入特殊字符：

| 转义序列 | 含义                  |
| :------- | :-------------------- |
| `\n`     | 换行                  |
| `\t`     | 制表符（Tab）         |
| `\\`     | 反斜杠本身            |
| `\'`     | 单引号                |
| `\"`     | 双引号                |
| `\r`     | 回车                  |
| `\uXXXX` | 16进制 Unicode 字符   |
| `\xhh`   | 8进制字符（十六进制） |

示例：

```py
print("第一行\n第二行")
print("列A\t列B")
print("显示一个反斜杠：\\")
print("Unicode 字符 ☺：\u263A")
```

如果希望字符串里的反斜杠就是普通字符（例如 Windows 路径、正则表达式），可以使用 **原始字符串**：

```python
path = r"C:\Users\Ada\new_folder"   # 不再把 \n 当作换行
regex = r"\d+\.\d+"
```

!!! note "注意"
    原始字符串的末尾不能是奇数个反斜杠，这会弄乱引号配对。解决办法是将末尾反斜杠写成双反斜杠，或使用普通字符串转义。

### 字符串的不可变性

字符串一旦创建，就不能修改其中的某个字符。如果真想得到一个改了某个字符的字符串，只能通过拼接、切片等方式生成一个新的字符串对象。

```py
s = "hello"
s[0] = "H"   # TypeError! 

new_s = "H" + s[1:]   # "Hello"
```

这个特性也是字符串能被用作字典键的原因之一，后面会讲到。

## 字符串索引与切片操作

> Python 将字符串视为一个有序的字符序列，可以像操作列表那样精确地取出单个字符或子串。

### 索引

索引从 0 开始，这是编程世界的惯例。

```py
s = "Python"
s[0]   # 'P'
s[1]   # 'y'
s[5]   # 'n'
```

负数索引从末尾开始计数，-1 是最后一个字符，-2 是倒数第二个，以此类推。

```py
s[-1]   # 'n'
s[-3]   # 'h'
```

使用超出范围的索引会引发 IndexError。空字符串没有有效的索引。

```py
s = ""
s[0]   # IndexError: string index out of range
```

### 切片

切片让你可以从一个字符串中 **抽出一段子串**，语法为 `[start:end:step]`。所有三个部分都是可选的。

- `start`：起始索引（包含），省略时默认为开头。
- `end`：结束索引（**不包含**），省略时默认为末尾。
- `step`：步长，可正可负，默认是 1。

```py
s = "Python"
s[0:3]     # 'Pyt'  索引0,1,2
s[:3]      # 'Pyt'  同上
s[2:]      # 'thon' 从索引2到末尾
s[:]       # 'Python' 完整拷贝
s[::2]     # 'Pto'  每隔一个字符取一个
s[::-1]    # 'nohtyP' 步长为负，字符串反转
```

切片非常宽容：即使索引超出范围，也不会报错，解释器会自动将起止点限制在有效区间内。

```py
s[2:100]   # 'thon' 安全
```

### 成员检测与长度

使用 `in` 关键字可以检查一个子串是否存在于字符串中：

```python
"th" in "Python"    # True
"xy" not in "Python"  # True
```

`len()` 函数返回字符串的字符个数，包括空格、符号和 Unicode 字符：

```python
len("Python")       # 6
len("你好，世界")    # 5 （Unicode字符串按字符计）
```

## 字符串格式化：%、format()、f-string

> 把变量值嵌入字符串是输出信息的关键。Python 有三种主流方法，按历史顺序从旧到新，推荐采用的也是越来越新。

### % 运算符（旧式）

风格来自 C 语言的 printf，使用 % 占位符。常用格式符：%s（字符串）、%d（整数）、%f（浮点数）、%x（十六进制）。若只有一个值，右侧可省略元组括号：

```py
name = "Ada"
age = 25
"我叫 %s，今年 %d 岁。" % (name, age)
# 结果: '我叫 Ada，今年 25 岁。'
"Pi is approximately %f" % 3.14
```

由于可读性较差且容易类型不匹配，新代码中应避免使用。

### str.format() 方法

引入于 Python 2.6，使用花括号 {} 作为占位符，通过 format() 方法传值。支持按位置或按名称引用参数，还可进行格式微调：

```py
"我叫 {}，今年 {} 岁。".format(name, age)

"圆周率：{:.2f}".format(3.14159)      # '圆周率：3.14'
"{name} 的分数是 {score}".format(name="Bob", score=95)
```

比 % 清晰灵活很多，但仍有些啰嗦。

### f-string（格式化字符串字面量）

Python 3.6 引入的 f-string，是目前最推荐的方式。在字符串前加 f 或 F，直接在花括号内写变量名甚至表达式。

```py
name = "Ada"
age = 25
f"我叫 {name}，明年我 {age + 1} 岁。"
# '我叫 Ada，明年我 26 岁。'
```

同样支持格式说明符：

```py
value = 1/3
f"结果: {value:.4f}"   # '结果: 0.3333'
```

!!! note "重要"
    f-string 不仅写起来最短，运行速度也最快，且语法错误能在 IDE 中被提前发现。目前唯一需要注意的是：它要求 Python 版本 ≥ 3.6。除非维护遗留项目，否则一律用 f-string。

## 类型转换与类型检查

> Python 是强类型语言，不同类型间的操作通常不会自动进行隐式转换，需要程序员显式调用。

### 类型转换函数

三个最常用的构造器：

- `int(x)`：将 x 转换为整数。可接受数字或纯数字字符串，可指定进制基数。

- `float(x)`：将 x 转换为浮点数。可接受数字或纯数字字符串。

- `str(x)`：将 x 转换为字符串。几乎所有对象都能转为字符串形式。

```py
int("42")         # 42
int("1010", 2)    # 10
float("3.14")     # 3.14
str(100)          # "100"
```

此外还有 `bool(x)`，`list(s)`，`tuple(s)` 等，分别对应不同的类型构造。
!!! note "注意"
    bool 的转换规律：`0`、`0.0`、空容器（`""`、`[]`、`{}`）、`None` 被视为 `False`，其余为 `True`。

### 类型检查

有两种常用方式，它们有细微但重要的区别。

**`type()` 函数**
返回对象的精确类型。

```py
x = 42
type(x) == int    # True
type(x) == str    # False
```

!!! warning "注意"
    `type()` 忽略继承关系。假如类 `B` 继承自类 `A`，`type(b_obj)` 只会告诉你它是 `B`，不会认为是 `A`。这在面向对象中有局限。

**`isinstance()` 函数**
检查对象是否是某个类的实例，且会考虑继承。

```python
x = 42
isinstance(x, int)    # True
isinstance(x, float)  # False
isinstance(True, int) # True，因为 bool 是 int 的子类
```

在需要处理多种可能的输入类型时，`isinstance()` 通常比 `type()` 更稳健。

```py
def add(a, b):
    if not (isinstance(a, (int, float)) and isinstance(b, (int, float))):
        raise TypeError("只支持数字类型")
    return a + b
```

### 何时转换，何时检查

基本原则：**信任输入，在边界处检查**。如果你编写的函数明确要求某种类型，可以在开头用 `isinstance` 做一层防御。内部代码尽量依赖 Python 的“鸭子类型”——只要对象的行为正确，不一定要用 `type()` 严格比对。这能保持代码的灵活优雅。

## 标准输入输出：input() 与 print()

### 获取用户输入：input()

`input()` 函数暂停程序的执行，等待用户在终端输入一行文字并按回车。然后它会将那行文字以 **字符串** 形式返回。

```python
name = input("请输入你的名字：")
print(f"你好，{name}！")
```

!!! note "重要提示"
    `input()` **永远返回字符串**，即使用户输入的是数字也如此：
    ```python
    age = input("年龄：")    # 用户输入 25
    type(age)               # <class 'str'>
    ```
    如果需要数字来进行计算，必须用 `int()` 或 `float()` 进行转换：
    ```python
    birth_year = int(input("出生年份："))
    current_year = 2026
    age = current_year - birth_year
    print(f"大约 {age} 岁")
    ```
    如果用户输入的不是纯数字，程序会崩溃并抛出 `ValueError`，异常处理将在第八章详述。

### 展示输出：print()

`print()` 是学习 Python 的第一个函数，它远比你想象的要灵活。它的完整签名大致为：

```python
print(*objects, sep=' ', end='\n', file=None, flush=False)
```

大多数情况下我们只传要打印的内容，此时多个参数会由 `sep` 指定的分隔符（默认是空格）连接，末尾自动加上换行符 `end`。

```python
print("Hello", "World")   # Hello World
```

**定制分隔符**：

```python
print("2026", "05", "11", sep="-")   # 2026-05-11
```

**取消换行**：

```python
print("Loading", end="")
print("...")   # 输出 Loading... 在同一行
```

`end` 参数常见用法：打印进度条或让循环输出不换行。

```python
for i in range(5):
    print(i, end=" ")   # 0 1 2 3 4 （最后有一个空格和换行）
```

**输出到文件**：

```python
with open("log.txt", "w") as f:
    print("程序启动", file=f)
```

!!! tip
    `print()` 本质上就是将对象的字符串表示写入 `file` 参数指定的文件对象，默认是 `sys.stdout`（标准输出，即终端）。掌握了这些参数，你就能精确控制输出格式。
