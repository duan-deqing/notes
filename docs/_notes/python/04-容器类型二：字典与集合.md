# 容器类型二：字典与集合

## 字典的创建与键值对规范

字典（dict）是 Python 中最重要的映射类型。它不像列表那样按位置管理数据，而是通过 **唯一的键** 来快速查找对应的值。可以把它理解为现实中的字典：根据一个词（键）查找它的释义（值）。

### 创建字典

最直观的方式是用花括号 `{}` 包裹 **键值对**，键和值之间用冒号分隔，每个键值对之间用逗号隔开。

```python
person = {
    "name": "Ada", 
    "age": 25, 
    "city": "Hangzhou"
}
empty_dict = {}
```

也可以用内置函数 `dict()` 从可迭代对象或关键字参数创建字典。

```python
# 从包含二元组（键,值）的可迭代对象创建
pairs = [
    ("name", "Ada"), 
    ("age", 25)
]
d1 = dict(pairs)   # {"name": "Ada", "age": 25}

# 使用关键字参数创建（键必须是合法的标识符）
d2 = dict(name="Ada", age=25)   # 同上
```

### 键的规范

字典的键必须是 **不可变且可哈希** 的类型。目前学过的类型中，**数字、字符串、元组（元组内的元素也必须可哈希）** 都可以作为键；而列表、字典、集合等可变类型不行。

```python
valid_keys = {
    1: "integer", 
    "pi": 3.14, 
    (1, 2): "tuple"
}

invalid_key = {
    [1, 2]: "list"
}   # TypeError: unhashable type: 'list'
```

值可以是任意 Python 对象，没有任何限制。

### 去重性

在同一个字典中，**键是唯一的**。如果创建时出现了重复的键，后出现的值会覆盖前面的值。

```python
d = {"a": 1, "b": 2, "a": 999}
print(d)   # {'a': 999, 'b': 2}
```

字典从 Python 3.7 开始 **保证保持插入顺序**（即遍历时按键添加的先后顺序出现）。这意味着它兼具了顺序容器的特点和映射的高效查找。

!!! warning "空字典与空集合的坑"
    `{}` 创建的是空字典，不是空集合。要创建空集合，必须使用 `set()`。这是初学者常踩的坑，后面 4.4 节会详细解释。

## 字典的访问、修改与删除

### 访问值

最直接的方式是用方括号加上 **键名**，但如果键不存在会抛出 `KeyError`。

```python
person = {"name": "Ada", "age": 25}
print(person["name"])   # 'Ada'
print(person["city"])   # KeyError: 'city'
```

**安全的访问方式** 是使用 `get()` 方法。它接受两个参数：键和一个可选的默认值。如果键存在，返回对应的值；否则返回默认值（默认为 `None`），不会引发异常。

```python
person.get("name", "unknown")  # 'Ada'
person.get("city", "unknown")  # 'unknown'
```

另一种安全访问是先用 `in` 检查成员。

```python
if "city" in person:
    print(person["city"])
```

### 修改与添加

字典是可变的，可以直接通过赋值来修改已有的键值对，或添加新的键值对。

```python
person["age"] = 26            # 修改已有键的值
person["job"] = "Engineer"    # 插入新键值对
```

如果想批量更新，可以使用 `update()` 方法，传入另一个字典或键值对可迭代对象。

```python
person.update({"age": 27, "city": "Beijing"})  # 已存在的修改，新键添加
```

!!! tip
    `setdefault(key, default)` 是一个有用的小工具：如果键存在，返回其值；如果键不存在，插入键并设为默认值，然后返回默认值。
    ```python
    counts = {}
    counts.setdefault("a", 0)    # 'a' 不存在，设为0，返回0
    counts["a"] += 1
    ```

### 删除键值对

- `pop(key, default)`：删除指定键并返回它对应的值；如果键不存在且未提供默认值，会引发 `KeyError`。提供默认值则会返回默认值。
- `popitem()`：移除并返回最后一个插入的键值对（Python 3.7+ LIFO 顺序）。空字典调用会 `KeyError`。
- `del` 语句：`del d[key]`，键不存在时 `KeyError`。
- `clear()`：清空整个字典。

```python
d = {"a": 1, "b": 2, "c": 3}
val = d.pop("b")        # val=2, d变为{"a":1, "c":3}
item = d.popitem()      # item=('c',3)
del d["a"]              # d变为{}
d.clear()
```

## 字典的遍历：keys / values / items

字典是可迭代对象，默认迭代的是它的 **键**。

```python
d = {"name": "Ada", "age": 25}
for key in d:
    print(key)   # name, age
```

但通常我们需要同时访问键和值，可以使用三个视图方法：

- `keys()`：返回所有键的视图。
- `values()`：返回所有值的视图。
- `items()`：返回所有键值对的视图，每个元素是 `(key, value)` 元组。

视图对象会 **动态反映字典的变化**，并且支持成员检测 `in`。通常我们会立即遍历它们。

```python
# 遍历键
for key in d.keys():
    print(key)

# 遍历值
for value in d.values():
    print(value)

# 同时遍历键和值（最常用）
for key, value in d.items():
    print(f"{key}: {value}")
```

其中 `items()` 结合解包，能让代码无比清晰。上面最后一个循环是处理字典数据的标准范式。

如果你需要一个真正的键列表（静态副本），可以用 `list(d.keys())`，但多数遍历场景直接用视图即可。

## 集合的创建与自动去重特性

集合（set）是一种 **无序、不重复** 的容器，元素必须是可哈希的（即不可变类型）。它的核心价值在于成员检测、去重和数学集合运算。

### 创建集合

- 用花括号 `{}` 包围元素，元素间逗号分隔（**注意：空花括号 `{}` 创建的是空字典！**）。
- 用 `set()` 构造函数，可以传入任何可迭代对象。

```python
fruits = {"apple", "banana", "cherry"}
empty_set = set()                # 这才是空集合
from_list = set([1, 2, 2, 3])   # {1, 2, 3}  自动去重
from_string = set("hello")       # {'h', 'e', 'l', 'o'}
```

### 无序性

集合不记录元素的插入顺序，因此不能通过索引访问。每次打印的顺序可能不同。

```python
s = {3, 1, 4, 1, 5}
print(s)   # 可能 {1, 3, 4, 5} 或别的顺序
```

尽管不能索引，但可以遍历，也可以使用 `in` 极快速地检测成员。

### 去重特性

如果从包含重复元素的列表或元组创建集合，重复值会被自动剔除。这是极简的去重方式，但代价是丢失原有顺序。若想保留顺序并去重，可借助字典的键（Python 3.7+）或手动编写逻辑。

```python
items = [2, 1, 2, 3, 1, 4]
unique = list(set(items))   # [1, 2, 3, 4] 顺序随机
# 保序去重（利用dict的键有序）
unique_ordered = list(dict.fromkeys(items))  # [2, 1, 3, 4]
```

### 单个元素操作

- `add(elem)`：添加一个元素，如果已存在则无变化。
- `remove(elem)`：移除元素，若不存在则 `KeyError`。
- `discard(elem)`：移除元素，若不存在不会报错。
- `pop()`：随机移除并返回一个元素，空集合会 `KeyError`。
- `clear()`：清空集合。

```python
s = {1, 2}
s.add(3)        # {1,2,3}
s.discard(4)    # 无变化，不报错
s.remove(1)     # {2,3}
```

## 集合运算：交集、并集、差集

集合支持丰富的数学集合运算，既可以用方法，也可以用运算符。运算符通常要求两个操作数都是集合，方法则可以接受任何可迭代对象。

设 `a = {1, 2, 3, 4}`，`b = {3, 4, 5, 6}`。

| 运算     | 运算符  | 方法                        | 描述                       |
| :------- | :------ | :-------------------------- | :------------------------- |
| 并集     | `a | b` | `a.union(b)`                | 包含所有不重复的元素         |
| 交集     | `a & b` | `a.intersection(b)`         | 两者都有的元素             |
| 差集     | `a - b` | `a.difference(b)`           | 在 a 但不在 b 的元素       |
| 对称差集 | `a ^ b` | `a.symmetric_difference(b)` | 在 a 或 b 中但不同时在两者 |

```python
a | b    # {1, 2, 3, 4, 5, 6}
a & b    # {3, 4}
a - b    # {1, 2}
a ^ b    # {1, 2, 5, 6}
```

还有对应的 **原地更新版本**，直接修改左侧集合：

- `a |= b` 或 `a.update(b)`
- `a &= b` 或 `a.intersection_update(b)`
- `a -= b` 或 `a.difference_update(b)`
- `a ^= b` 或 `a.symmetric_difference_update(b)`

### 子集与超集判断

- `a.issubset(b)` 或 `a <= b`：a 是否是 b 的子集。
- `a.issuperset(b)` 或 `a >= b`：a 是否是 b 的超集。
- `<` 和 `>` 用于真子集、真超集（排除了相等的情况）。

```python
{1, 2} <= {1, 2, 3}   # True
{1, 2} < {1, 2}        # False
```

!!! note "集合的应用场景"
    - 快速去重。
    - 成员存在性检查（O(1) 平均时间）。
    - 高效处理大量数据的交叉、合并问题（如共同好友、标签系统）。

## 推导式：列表、字典与集合推导式

推导式（comprehension）是 Python 的一颗语法糖，它用极简的表达式从一个可迭代对象生成新的容器，往往一行能顶数行循环。

### 列表推导式

基本形式：

```python
[expression for item in iterable if condition]
```

- `expression` 是对 `item` 的运算，生成新列表的元素。
- `for item in iterable` 是遍历元素。
- `if condition` 是可选过滤。

实例：

```python
# 生成平方数
squares = [x**2 for x in range(10)]
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# 带过滤：只保留偶数平方
even_squares = [x**2 for x in range(10) if x % 2 == 0]
# [0, 4, 16, 36, 64]

# 可嵌套多个for（从左到右顺序）
pairs = [(x, y) for x in [1,2,3] for y in [3,1,4] if x != y]
# [(1,3), (1,4), (2,3), (2,1), (2,4), (3,1), (3,4)]
```

列表推导式执行速度快于等价的 `for` 循环，因为它是在 C 层面实现循环，但仍要注意可读性：嵌套过多或逻辑复杂时，拆成普通循环可能更清晰。

### 字典推导式

语法与列表推导式类似，但用花括号且使用 `key: value` 表达式。

```python
{key_expr: value_expr for item in iterable if condition}
```

实例：

```python
# 数字到平方的映射
square_map = {x: x**2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# 反转字典的键值（假设值都是唯一的）
original = {"a": 1, "b": 2}
reversed_dict = {v: k for k, v in original.items()}
# {1: 'a', 2: 'b'}

# 带过滤
even_map = {x: x**2 for x in range(10) if x % 2 == 0}
```

### 集合推导式

也使用花括号，但只产生单个元素表达式，没有冒号。

```python
{expression for item in iterable if condition}
```

实例：

```python
# 所有字母的 ASCII 码集合
codes = {ord(c) for c in "hello"}
# {104, 101, 108, 111, 111} 去重后为 {104, 101, 108, 111}

# 从列表去重并保留平方大于5的值
nums = [1, 2, 2, 3, 4]
result = {x**2 for x in nums if x**2 > 5}
# {9, 16}
```

### 生成器表达式

当需要惰性计算而不是立即生成整个容器时，可以去掉外部方括号、花括号，只保留小括号，这就变成了 **生成器表达式**。它不一次性占用大量内存，而是一次产生一个元素。生成器的细节在第十一章详述。

```python
gen = (x**2 for x in range(1000000))   # 只创建生成器对象，内存占用极小
```

## 本章小结

字典和集合是 Python 中处理关联数据和唯一性需求的两大利器。字典以键值对的形式提供了高效查找，而集合则借助哈希表实现了去重和数学集合运算。推导式用声明式的语法让你快速构建这些容器，写起来流畅，读起来也像数学描述。至此，你已经掌握了 Python 四种最核心的内置容器：列表、元组、字典、集合。接下来，我们将学习如何用条件判断和循环让程序拥有决策与重复执行的能力。
