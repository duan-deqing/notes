    === MAIN CONTENT ===
    Use simple hand-drawn cartoon doodles (marker sketch style)

    [CONTENT_SECTION_1]
    Character/Element: [ELEMENT_NAME]

    Expression/State: exaggerated marker doodle expression

    Props: simple outlined objects with red/blue accent lines
    Visual: sketchy icon with motion lines
    Label: "[LABEL_TEXT]" written as if quickly handwritten
    [DIALOGUE_OR_ANNOTATION] in speech bubble

    [CONTENT_SECTION_2]
    [Similar structure as above]

    [CONTENT_SECTION_N]
    [Continue as needed]

    === KEY CONCEPTS ===
    [NUMBER] floating info boxes drawn as marker rectangles with uneven edges:
    Box 1: "[CONCEPT_1]"
    Box 2: "[CONCEPT_2]"
    Box N: "[CONCEPT_N]"

    === WHITEBOARD STYLE ===
    Real classroom whiteboard aesthetic, black base marker lines, red and blue emphasis strokes, visible stroke overlap, slight perspective depth, natural lighting, casual sketch composition, instructional yet playful

    === MOOD ===
    Energetic, explanatory, slightly humorous, practical and easy to understand

    Quality: High-resolution whiteboard illustration, clear visual hierarchy, suitable for educational thumbnail or knowledge-sharing poster
